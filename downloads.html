Windows

The Windows build has started downloading. When it finishes, extract the archive and launch ByAThread.exe. If SmartScreen warns you, choose More info and then Run anyway.

macOS Apple Silicon

The Apple Silicon build is downloading. Open the downloaded .dmg, drag the game into Applications, then launch it. If macOS blocks it, open System Settings, Privacy and Security, and allow the app.

macOS Intel

The Intel build is downloading. Open the downloaded .dmg, copy the game to Applications, and start it from there. If Gatekeeper blocks it, allow it from Privacy and Security.

Linux AppImage

The AppImage download has started. Mark it executable before launching: chmod +x ByAThread.AppImage. Then run it directly from your file manager or terminal.

Linux .deb

The .deb package is downloading. Install with your package manager or run: sudo apt install ./by-a-thread.deb, then launch the game from your app menu.

Linux .rpm

The .rpm package is downloading. Install it with your distro tools, for example: sudo dnf install ./by-a-thread.rpm, and launch from your applications menu.
